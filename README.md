#NUMBER GUESSER
Welcome to NumberGuesser, the name is self explanatory, this is an interactive game where the computer randomly selects a number within the defined range and YOU the player will try to guess it. After each guess the program will give you a feedback such as whether the guess is too high, too low, or correct. Have fun guessing!

#ONLYREALTRIANGLESALLOWED
Welcome to OnlyRealTrianglesAllowed, this program validates real triangles, first the program asks the user for three positive side lengths and checks if they can form a valid triangle/dorito.
